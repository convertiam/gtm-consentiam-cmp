# Consentiam CMP — Google Tag Manager Template

Cookie consent banner with Google Consent Mode v2 support. GDPR & ePrivacy compliant.

## Installation

1. In Google Tag Manager, go to **Templates → Search Gallery**
2. Search for **"Consentiam CMP"**
3. Add to workspace
4. Create a new tag with trigger **Consent Initialization — All Pages**
5. Enter your Consentiam API Key from [your dashboard](https://www.consentiam.eu/dashboard)
6. Save and publish
